Panda autograsp module¶

panda_autograsp.functions

This modules contains some additional functions which are not part of a specific module.

panda_autograsp.grasp_planner

Implementation of the GQCNN grasp detection algorithm on the Franka Emika Panda Robots.This file is based on the grasp_planner_node.py file that was supplied with the GQCNN package..

Panda autograsp Tools¶

tools.plan_grasp

Implementation of the GQCNN grasp detection algorithm on the Franka Emika Panda Robots.This file is based on the grasp_planner_node.py file that was supplied with the GQCNN package..

tools.chessboard_calibration

Simple calibration script created with the opencv2 module.

tools.test_perception_calibration

Script to test autolab calibration capabilities.
