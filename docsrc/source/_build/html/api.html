Scripts Module¶

Implementation of the GQCNN grasp detection algorithm on the Franka Emika Panda Robots. This file is based on the grasp_planner_node.py file that was supplied with the GQCNN package.

class scripts.grasp_planner.GQCNNGrasp¶

Class for storing the computed grasps. This class is a trimmed down version of the original gqcnn.grasping.policy.policy.GraspAction.

property PARALLEL_JAW¶

PARALLEL_JAW Specifies parallel jaw gripper type information

property SUCTION¶

SUCTION Specifies suction jaw gripper type information

class scripts.grasp_planner.GraspPlanner(cfg, grasping_policy, sensor_type='kinectv2')¶

GraspPlanner class that interacts with a number of grasp detection algorithms to compute the optimal grasp. Currently only the GQCNN by berkeleyautomation is supported.

execute_policy(rgbd_image_state, grasping_policy, pose_frame)¶

Executes a grasping policy on an RgbdImageState.

Parameters

  • rgbd_image_state (RgbdImageState) – RgbdImageState from BerkeleyAutomation/perception to encapsulate depth and color image along with camera intrinsics.

  • grasping_policy (GraspingPolicy) – Grasping policy to use.

  • pose_frame (str) – Frame of reference to publish pose in.
